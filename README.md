# Geometric-Transformations-Perspective-Correction-using-OpenCV
در این پروژه با استفاده از کتابخانه‌ی OpenCV، مجموعه‌ای از تبدیلات هندسی روی تصویر انجام می‌شود.
هدف اصلی این پروژه بررسی و پیاده‌سازی عملی مفاهیم فصل پنجم (تبدیلات هندسی، پرسپکتیو و اعوجاج) است.

ورودی پروژه یک تصویر واقعی است که دارای اعوجاج پرسپکتیو می‌باشد (مانند تصویر برج ساعت). سپس با انتخاب یک ناحیه‌ی مشخص از تصویر (با انتخاب ۴ گوشه توسط کاربر)، چندین تبدیل هندسی روی آن اعمال می‌گردد و نتایج در قالب تصاویر خروجی ذخیره می‌شوند.




هدف این پروژه:

پیاده‌سازی انواع تبدیل‌های هندسی (Affine & Projective)

اصلاح پرسپکتیو یک بخش از تصویر با استفاده از انتخاب ۴ نقطه (Homography)

بررسی اثر انواع روش‌های درون‌یابی (Interpolation)

انجام مرحله‌ی رفع اعوجاج (Undistortion) به‌صورت تقریبی

بهبود کیفیت تصویر با فیلترهای اضافی جهت افزایش امتیاز پروژه





تصویر ورودی و مشکل آن

تصویر استفاده شده یک عکس واقعی از یک برج ساعت است که به‌صورت زاویه‌دار گرفته شده و دارای:

اعوجاج پرسپکتیو (Perspective Distortion)
به دلیل زاویه دید دوربین، خطوط موازی در واقعیت در تصویر همگرا دیده می‌شوند




الگوریتم‌ها و روش‌های استفاده شده
1) انتخاب ۴ نقطه (Manual Corner Selection)

کاربر با کلیک موس چهار گوشه یک ناحیه‌ی موردنظر (مثل قاب ساعت) را مشخص می‌کند.

2) تبدیل پرسپکتیو (Perspective Transform / Homography)

برای تبدیل ناحیه انتخاب شده به یک نمای روبه‌رو از این روش استفاده شده است:

cv2.getPerspectiveTransform()

cv2.warpPerspective()

این تبدیل باعث می‌شود یک ناحیه‌ی مایل (کج) به صورت مستقیم و صاف بازسازی شود.

3) درون‌یابی (Interpolation)

در مراحل تبدیل پرسپکتیو و تبدیل‌های affine، اثر روش‌های مختلف درون‌یابی بررسی می‌شود:

Nearest Neighbor (INTER_NEAREST)

Bilinear (INTER_LINEAR)

Bicubic (INTER_CUBIC)

Lanczos (INTER_LANCZOS4)

هدف این بخش مشاهده تفاوت کیفیت خروجی در روش‌های مختلف نمونه‌برداری است.

4) رفع اعوجاج (Approximate Undistortion)

رفع اعوجاج دوربین به صورت تقریبی با cv2.undistort() انجام شده است.

چون اطلاعات کالیبراسیون دوربین در دسترس نیست، این مرحله به شکل تقریبی انجام می‌شود و اثر آن در تصاویر با اعوجاج کم ممکن است زیاد محسوس نباشد.

5) تبدیل‌های هندسی Affine

پس از اصلاح پرسپکتیو، تبدیل‌های پایه‌ی هندسی روی تصویر اعمال می‌شود:

 انتقال (Translation) → cv2.warpAffine()
 چرخش (Rotation) → cv2.getRotationMatrix2D() + warpAffine()
 مقیاس‌گذاری (Scaling) → warpAffine()
کج کردن (Shear / Skew) → warpAffine()

6) فیلترهای اضافی جهت افزایش کیفیت (Extra Filters)

برای افزایش کیفیت و امتیاز پروژه، چند فیلتر نیز به خروجی اعمال می‌شود:

CLAHE (افزایش کنتراست محلی)

Denoising (کاهش نویز با NLM)

Sharpening (افزایش وضوح)
