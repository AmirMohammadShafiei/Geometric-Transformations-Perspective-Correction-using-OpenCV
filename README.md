# Geometric-Transformations-Perspective-Correction-using-OpenCV
در این پروژه با استفاده از کتابخانه‌ی OpenCV، مجموعه‌ای از تبدیلات هندسی روی تصویر انجام می‌شود.
هدف اصلی این پروژه بررسی و پیاده‌سازی عملی مفاهیم فصل پنجم (تبدیلات هندسی، پرسپکتیو و اعوجاج) است.

ورودی پروژه یک تصویر واقعی است که دارای اعوجاج پرسپکتیو می‌باشد (مانند تصویر برج ساعت). سپس با انتخاب یک ناحیه‌ی مشخص از تصویر (با انتخاب ۴ گوشه توسط کاربر)، چندین تبدیل هندسی روی آن اعمال می‌گردد و نتایج در قالب تصاویر خروجی ذخیره می‌شوند.
